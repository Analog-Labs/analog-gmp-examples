# Teleport Tokens

This example demonstrates how teleport an ERC20 from Alice's account in Shibuya to Bob's account in Sepolia.

### Prerequisite

-   [Install Forge](https://book.getfoundry.sh/getting-started/installation)

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```
